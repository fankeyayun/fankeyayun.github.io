添加评论功能

发表于 2019-12-13 分类于 hexo Valine： 155 1 分钟
添加评论是采用了默认的Valine框架(无后端)，简洁非常方便。

  1. 首先要到LeanCloud注册要它的appid和appkey。

  2. 其次点击安全中心设置安全域名(你的域名)

  3. 在hexo 主题配置件_config.yml 找到 并开启评论 true,同时配上刚从LeanCoud拿到的appid和appkey。

    详情配置文档

  4. 最后 hexo clean hexo server访问本地就能够看到初始的评论
